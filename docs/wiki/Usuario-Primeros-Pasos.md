# Primeros Pasos en Ceiba

Esta guía te ayudará a comenzar a usar el sistema Ceiba para la gestión de reportes de incidencias.

## Acceder al Sistema

### 1. Abrir la Aplicación

Abre tu navegador web (Chrome, Firefox, Edge o Safari) y navega a la dirección del sistema que te proporcionó tu administrador.

### 2. Iniciar Sesión

En la pantalla de inicio de sesión:

1. Ingresa tu **Correo Electrónico** institucional
2. Ingresa tu **Contraseña** (mínimo 10 caracteres)
3. Opcionalmente, marca "Recordar sesión" para mantener tu sesión activa
4. Haz clic en **Iniciar Sesión**

![Pantalla de login](images/login-pantalla-principal.png)

> **Nota:** Si olvidaste tu contraseña, contacta al administrador del sistema para restablecerla.

### 3. Pantalla de Inicio

Después de iniciar sesión, verás la pantalla de inicio con las opciones disponibles según tu rol:

- **CREADOR**: Panel de Reportes para crear y gestionar tus propios reportes
- **REVISOR**: Panel de Supervisión para ver todos los reportes y exportar
- **ADMIN**: Panel de Administración para gestionar usuarios y catálogos

![Dashboard principal](images/creador-dashboard.png)

## Información de tu Sesión

En la parte inferior de la pantalla de inicio puedes ver:

- **Usuario**: Tu correo electrónico
- **Roles**: Los roles que tienes asignados
- **Sesión expira**: La sesión se cierra automáticamente después de 30 minutos de inactividad

## Cerrar Sesión

Para cerrar tu sesión de forma segura:

1. Haz clic en tu nombre de usuario en la barra superior
2. Selecciona **Cerrar Sesión**

> **Advertencia:** Siempre cierra sesión cuando termines de usar el sistema, especialmente en equipos compartidos.

## Navegación Básica

### Barra Superior

La barra superior contiene:
- Nombre de la aplicación (Ceiba)
- Menú de navegación
- Tu nombre de usuario y opción para cerrar sesión

### Menú Lateral

En dispositivos móviles, el menú se oculta y aparece un ícono de hamburguesa para expandirlo.

Si tienes problemas para acceder al sistema:

1. Verifica que tu correo electrónico esté escrito correctamente
2. Asegúrate de que tu cuenta esté activa
3. Contacta al administrador del sistema si el problema persiste
