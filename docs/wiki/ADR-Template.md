## Plantilla de Uso

Para crear un nuevo ADR:

1. Copia este template
2. Renombra a `ADR-XXX-nombre-descriptivo.md`
3. Reemplaza los placeholders con información real
4. Somete a revisión del equipo
5. Actualiza el estado según el resultado

## Nomenclatura

- **ADR-001 a ADR-099**: Decisiones de arquitectura de sistema
- **ADR-100 a ADR-199**: Decisiones de tecnología/librerías
- **ADR-200 a ADR-299**: Decisiones de infraestructura/devops
- **ADR-300 a ADR-399**: Decisiones de procesos/metodología
