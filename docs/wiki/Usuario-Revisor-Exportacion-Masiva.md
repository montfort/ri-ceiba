# Exportación Masiva de Reportes

La exportación masiva te permite generar archivos con múltiples reportes de forma eficiente.

## Cuándo Usar Exportación Masiva

- Generar reportes periódicos (diarios, semanales, mensuales)
- Crear archivos de respaldo
- Preparar información para reuniones
- Enviar datos a otras áreas

## Acceder a la Exportación Masiva

1. Ve a **Exportar** desde el panel principal
2. Usa la sección principal para selección manual
3. O usa **Exportación Rápida** para períodos predefinidos

![Exportación masiva](images/revisor-exportar-masivo.png)

## Exportación Rápida

### Reportes de Hoy

Exporta todos los reportes creados en el día actual:

1. En la sección "Exportación Rápida"
2. Encuentra la tarjeta **Reportes de Hoy**
3. Haz clic en **PDF** o **JSON**
4. La exportación se genera automáticamente

### Últimos 7 Días

Exporta reportes de la última semana:

1. Encuentra la tarjeta **Últimos 7 Días**
2. Selecciona el formato deseado
3. Descarga el archivo generado

### Este Mes

Exporta todos los reportes del mes actual:

1. Encuentra la tarjeta **Este Mes**
2. Selecciona PDF o JSON
3. El archivo incluirá todos los reportes desde el día 1 del mes

## Exportación Manual con Selección

Para más control sobre qué reportes incluir:

### Paso 1: Configurar Filtros

1. Selecciona el **Estado** (o deja en "Todos")
2. Especifica el **Delito** si necesitas filtrarlo
3. Define el rango de **fechas**

### Paso 2: Buscar

1. Haz clic en **Buscar Reportes**
2. Revisa los resultados en la tabla

### Paso 3: Seleccionar Todo

1. Marca la casilla **Seleccionar todos**
2. Todos los reportes mostrados quedarán seleccionados
3. El contador mostrará el total

### Paso 4: Exportar

1. Haz clic en **Exportar PDF** o **Exportar JSON**
2. Espera la generación (puede tomar tiempo para muchos reportes)
3. Descarga el archivo

## Límites de Exportación

| Parámetro | Límite |
|-----------|--------|
| Reportes por exportación | 500 máximo |
| Tamaño del archivo PDF | ~100MB |
| Tiempo de generación | Hasta 2 minutos |

### Si tienes más de 500 reportes

1. Divide la exportación en rangos de fecha
2. Aplica filtros adicionales
3. Realiza múltiples exportaciones

## Gestión de Selección

### Seleccionar Todos

Marca la casilla superior para seleccionar todos los reportes mostrados.

### Deseleccionar Todo

Haz clic en **Limpiar selección** para quitar todas las marcas.

### Selección Individual

Haz clic en cada fila o casilla para seleccionar/deseleccionar reportes específicos.

## Indicadores de Progreso

Durante la exportación:
- Un spinner indica que se está procesando
- El botón se deshabilita para evitar duplicados
- Al completar, el archivo se descarga automáticamente

## Mensajes de Estado

| Mensaje | Significado |
|---------|-------------|
| "X reportes exportados exitosamente" | Exportación completada |
| "No hay reportes para el período" | No se encontraron datos |
| "Error al exportar" | Problema técnico |

## Consejos para Exportación Eficiente

1. **Usa filtros específicos**: Reduce el conjunto de datos
2. **Exporta por períodos**: Semanal o mensual es más manejable
3. **Verifica antes de exportar**: Revisa el conteo de reportes
4. **Organiza los archivos**: Nombra las descargas según su contenido
